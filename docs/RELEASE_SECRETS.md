# Release Secrets

## npm (public)

- `NPM_TOKEN`

## pub.dev (Flutter)

- `PUB_DEV_PUBLISH_ACCESS_TOKEN`
- `PUB_DEV_PUBLISH_REFRESH_TOKEN`
- `PUB_DEV_PUBLISH_TOKEN_ENDPOINT`
- `PUB_DEV_PUBLISH_EXPIRATION`

## Optional hardening

- `GH_PAT_RELEASE` (if using GitHub release APIs beyond default `GITHUB_TOKEN`)
